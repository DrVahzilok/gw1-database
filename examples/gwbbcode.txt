To specify an attribute level, you have to write enough of the attribute name for gwBBCode to understand which one it is. Case doesn't matter but don't use spaces. Actually, first three letters are often enough, but there are 3 exceptions: dea is Death Magic and dead is Deadly Arts; com is Communing and comma is Command; ear is Earth Magic and earthp is Earth Prayers.

Here 'pvp' can be used instead of 'pve', or you can specify whether the skill is PvP or PvE in its name: [b][pre]["Watch Yourself!" (PvP)][/pre][/b]

To display a skill you need to encase its name in double square brackets "[b][[[/b]" and "[b]]][/b]".
For example the skill Shock would be represented by
[pre][b][[Shock]][/b][/pre] => [[Shock]]
If you want to show the skill name instead of its icon, put a second "[b][[/b]" before the name
[pre][b][[Shock][/b][/pre] => [[Shock]
If you want to show a skill effect adjusted to an attribute value, include "@" after skill name and then a number
[b][pre][[Shock@8][/pre][/b] => [[Shock@8] here we have the skill with 8 in Air Magic
[i]Tip[/i]: case and punctuation don't matter in skill names, and you can use some abbreviations


[size=14][b][u]How to write a build[/u][/b][/size]

The simpler way is to get its template code from the game, then use it in one of these ways:
[pre][Shock Warr;OQYTgw4K5QrSFqM9cOtrCA]
[build=Shock Warr;OQYTgw4K5QrSFqM9cOtrCA][/pre]
[Shock Warr;OQYTgw4K5QrSFqM9cOtrCA]

But if you want to write a build from scratch, here is how! A build consists of an opening tag [pre][build][/pre] in which you can specify details (see below),
then a list of skills encased in brackets, and an ending tag [pre][/build][/pre]. In this example we specified a name and attribute level. Don't panic it will be explained in details below :)
[pre][build prof=W/E air=8][Shock][Frenzy][Eviscerate][/build][/pre]
[build prof=W/E air=8][Shock][Frenzy][Eviscerate][/build]

Now let's have a closer look.. at what you can specify in the [pre][build][/pre] tag..
[list]
[*][u]Professions:[/u] To specify the build is for a Warrior/Elementalist, just add [b]prof=W/E[/b].
You can use any profession abbreviation: W, Mo, E, R, N, Me, Rt, A, D, P. As well as full profession names, e.g prof=Warrior/Elementalist. You don't have to specify a secondary profession.
[b][pre][build prof=W/E][/build][/pre][/b] = [b][pre][build prof=Warrior/Elementalist][/build][/pre][/b] =>
[build prof=W/E][/build]

[*][u]Build's name:[/u] To specify a name, try this: [b]name="Shock war"[/b] (note: the name must be encased in double brackets [b]""[/b])
[b][pre][build prof=W/E name="Shock war"][/build][/pre][/b] =>
[build prof=W/E name="Shock war"][/build]

[*][u]Attributes:[/u] Just write the attribute name [i][u]without spaces[/u][/i], followed by "[b]=[/b]", and the attribute level. Like this: [b]expertise=9[/b]. You can use plus signs to specify additional levels due to a rune or helmet: [b]axemastery=12+1+3[/b].
[i]Tip[/i]: Case doesn't matter here and all but 3 attribute names can be reduced to their first three letters, e.g [b]axe=12[/b] and [b]pro=4[/b]. Specifying a few more letters doesn't hurt though: [b]prot=9[/b] and [b]blood=8[/b]. The three exceptions to this rule are: [b]dead[/b] for Deadly Arts since [b]dea[/b] is for Death Magic; [b]comma[/b] for Command since [b]com[/b] is for Communing; [b]earthp[/b] for Earth Prayers since [b]ear[/b] is for Earth Magic.
[b][pre][build prof=W/E AxeMastery=12+1+3][/build][/pre][/b] = [b][pre][build prof=W/E axe=12+1+3][/build][/pre][/b] =>
[build prof=W/E axe=12+1+3][/build]

[*][u]Description:[/u] Adding information on a build (weapons, armor, how to play it) can be done like this: [b]desc="Vamp axe."[/b] (note: the description must be encased in double brackets [b]""[/b])
[b][pre][build prof=W/E desc="Vamp axe."][/build][/pre][/b] =>
[build prof=W/E desc="Vamp axe. Spike with Shock."][/build]
[i]Tip[/i]: you can write descriptions in multiple lines using [b]{br}[/b] to separate lines. This tip works on build names as well.

[*][u]PvE/PvP:[/u] You can specify whether a build is PvE or PvP by adding either "pve" or "pvp" to the code:
[b][pre][build prof=W/E pve]["Watch Yourself!"][Blinding Surge][/build][/pre][/b] =>
[build prof=W/E pve]["Watch Yourself!"][Blinding Surge][/build]
[b][pre][build prof=W/E pvp]["Watch Yourself!"][Blinding Surge][/build][/pre][/b] =>
[build prof=W/E pvp]["Watch Yourself!"][Blinding Surge][/build]
[/list]
